# B_Looker – QR-Based Material & Machine Tracking System

## Overview

**B_Looker (Barrel Looker)** is a web-based Smart Manufacturing application developed for the **Barrel (Tumbling) process** in Lenskart's eyewear manufacturing operations.

The Barrel (Tumbling) process is used to smooth and polish eyewear frame components before further manufacturing. B_Looker digitizes this process by enabling QR-based machine and material tracking, automating production data collection, storing records in Google Sheets, and providing real-time production monitoring through an interactive dashboard.

---

## Features

- QR-based Machine & Material Scanning
- Automatic Data Validation
- Google Sheets Integration
- Real-Time Production Monitoring
- Mobile-Friendly Interface
- Progressive Web App (PWA)
- Manual Data Entry Support
- Digital Production Tracking

## Project Workflow

1. Scan Machine QR Code
2. Scan Material QR Code
3. Validate QR Data
4. Store Data in Google Sheets
5. Monitor Production through Dashboard
